# **App Name**: FileFlex

## Core Features:

- PDF Conversion: Convert PDFs to various formats (Word, JPG, TXT) and perform operations like merging and splitting.
- Image Conversion: Convert images between JPG, PNG, GIF, SVG formats, with resizing and compression options.
- Video Conversion: Convert videos between MP4, MOV, AVI, MKV formats, with compression and audio extraction features.
- Audio Conversion: Convert audio files between MP3, WAV, AAC, OGG formats and allow for bitrate adjustment.
- Drag & Drop Upload: Enable users to upload files by dragging and dropping them onto the designated area.
- File Handling: Automatic deletion of converted files from the server after conversion to ensure privacy. Implement error handling for unsupported file formats or files exceeding a size limit.
- Format Selection: Suggest the most efficient file conversion tool given a set of input features of the target filetype to convert into.

## Style Guidelines:

- Primary color: Deep purple (#673AB7) to convey a sense of professionalism and innovation.
- Background color: Light gray (#F5F5F5) for a clean and modern look.
- Accent color: Soft blue (#2196F3) to highlight interactive elements.
- Body and headline font: 'Inter', a grotesque-style sans-serif, for a modern and neutral feel suitable for both headlines and body text.
- Use simple, outline-style icons for file types and conversion actions.
- Implement a responsive grid layout for converters, ensuring a seamless experience on desktop and mobile devices.
- Add subtle animations for button interactions, progress updates, and file uploads to enhance user engagement.
